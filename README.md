🌡️ Temperature Converter (PHP)

A simple and responsive Temperature Converter Web App built using PHP and Bootstrap. This project allows users to convert temperatures between Celsius (°C) and Fahrenheit (°F) instantly.

🚀 Features
Convert Celsius to Fahrenheit
Convert Fahrenheit to Celsius
Input validation (accepts only numeric values)
Clean and responsive UI using Bootstrap 5
Instant result display after submission
🛠️ Technologies Used
PHP (Backend logic)
HTML5 (Structure)
CSS3 (Styling)
Bootstrap 5 (Responsive design)

⚙️ How It Works
User enters a temperature value.
Selects the conversion type:
Celsius → Fahrenheit
Fahrenheit → Celsius
Clicks on "Convert Now"
PHP processes the input and displays the result.
🧮 Conversion Formula

Celsius to Fahrenheit:

(C × 9/5) + 32

Fahrenheit to Celsius:

(F − 32) × 5/9
💻 Installation & Usage

Clone the repository:

git clone https://github.com/your-username/temperature-converter.git
Move the project to your server directory:
For XAMPP → htdocs
For WAMP → www
Start Apache server.

Open your browser and go to:

http://localhost/temperature-converter
